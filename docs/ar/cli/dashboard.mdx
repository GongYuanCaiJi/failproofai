---

```bash
failproofai
```

يشغّل لوحة التحكم على الويب في `http://localhost:8020`.

## الخيارات

| العلم | الوصف |
|------|-------------|
| `--port <number>` | المنفذ المراد الاستماع عليه (الافتراضي: `8020`) |
| `--allowed-origins <origins>` | قائمة مفصولة بفواصل من المضيفين/عناوين IP المسموح لها بالوصول إلى موارد التطوير |

لتوجيه لوحة التحكم إلى مجلد Claude مشروع غير افتراضي، قم بتعيين متغير البيئة `CLAUDE_PROJECTS_PATH` عند التشغيل.

## أمثلة

```bash
# شغّل على منفذ مختلف
failproofai --port 9000

# استخدم مسار مشاريع Claude مخصص عبر متغير البيئة
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
