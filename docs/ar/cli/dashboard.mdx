title: عرض الجلسات
description: "قم بتشغيل لوحة التحكم لاستعراض جلسات الوكيل وإدارة السياسات"
---

```bash
failproofai
```

يبدأ لوحة التحكم على الويب في `http://localhost:8020`.

## الخيارات

| العلم | الوصف |
|------|-------------|
| `--port <number>` | المنفذ الذي سيتم الاستماع عليه (الافتراضي: `8020`) |
| `--allowed-origins <origins>` | قائمة مضيفين/عناوين IP مفصولة بفواصل مسموح لها بالوصول إلى موارد التطوير |

لتوجيه لوحة التحكم إلى مجلد Claude project غير افتراضي، قم بتعيين متغير البيئة `CLAUDE_PROJECTS_PATH` عند التشغيل.

## أمثلة

```bash
# قم بالتشغيل على منفذ مختلف
failproofai --port 9000

# استخدم مسار Claude projects مخصص عبر متغير البيئة
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
