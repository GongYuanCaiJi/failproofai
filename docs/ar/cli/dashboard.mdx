---
title: عرض الجلسات
description: "شغّل لوحة المعلومات لاستعراض جلسات الوكلاء وإدارة السياسات"
---

```bash
failproofai
```

يشغّل لوحة المعلومات على الويب في `http://localhost:8020`.

## الخيارات

| العلم | الوصف |
|------|-------------|
| `--port <number>` | المنفذ المراد الاستماع عليه (الافتراضي: `8020`) |
| `--projects-path <path>` | تجاوز موقع مجلدات مشاريع Claude Code |
| `--allowed-origins <origins>` | مضيفون وعناوين IP مفصولة بفواصل مسموحة بالوصول إلى موارد التطوير |

## أمثلة

```bash
# شغّل على منفذ مختلف
failproofai --port 9000

# استخدم مسار مشاريع مخصص
failproofai --projects-path ~/my-claude-projects
```
