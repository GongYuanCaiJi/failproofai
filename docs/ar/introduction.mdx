خطافات وسياسات تحافظ على موثوقية وكلائك الذكيين وتعمل بشكل مستقل - لـ **Claude Code** و **OpenAI Codex** و **Agents SDK**.

الوكلاء الذكيون يفشلون بطرق يمكن التنبؤ بها. يقومون بتنفيذ أوامر تخريبية، وتسريب الأسرار، والانحراف عن المهام، والعلق في حلقات لا نهائية، أو الدفع مباشرة إلى الفرع الرئيسي. إذا تركتهم دون مراقبة، ستتحول الأخطاء الصغيرة إلى مشاكل كبيرة.

يحل Failproof AI هذه المشكلة بـ **السياسات** - قواعد تتدخل في كل استدعاء أداة وكيل ويمكنها **كشف الأخطاء** و **تخفيفها** (الحظر أو التعليمات أو التنقية) و **تنبيهك** عندما تحتاج شيء ما إلى اهتمام. لوحة معلومات محلية تتيح لك مراجعة كل شيء لاحقًا.

كل شيء يعمل محليًا. لا تترك أي بيانات جهازك.

## البدء السريع

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```
