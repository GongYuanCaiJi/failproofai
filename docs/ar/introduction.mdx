Hooks و policies تحافظ على موثوقية وكفاءة عمل وكلائك الذكيين بشكل مستقل - لـ **Claude Code** و **OpenAI Codex** و **Agents SDK**.

الوكلاء الذكيون يفشلون بطرق يمكن التنبؤ بها. يقومون بتنفيذ أوامر مدمرة أو يسربون الأسرار أو ينجرفون بعيداً عن المهمة المطلوبة أو يعلقون في حلقات لا نهائية أو يدفعون مباشرة إلى الفرع الرئيسي. إذا تُركوا بدون مراقبة، فإن الأخطاء الصغيرة تتسبب في فشل متسلسل.

Failproof AI يحل هذه المشكلة عبر **policies** - قواعد متصلة بكل استدعاء أداة للوكيل ويمكنها **كشف الأخطاء** و **التعامل معها** (الحظر أو الإرشاد أو التنظيف) و **إرسال إنبيهات** عند الحاجة إلى اهتمام. لوحة معلومات محلية تتيح لك مراجعة كل شيء بعد ذلك.

كل شيء يعمل محلياً. لا توجد بيانات تغادر جهازك.

## البدء السريع

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # تفعيل السياسات
failproofai                      # تشغيل لوحة المعلومات
```

اطلع على دليل [البدء السريع](/ar/getting-started) للاطلاع على الشرح الكامل.
