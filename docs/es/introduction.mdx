Hooks y políticas que mantienen a tus agentes de IA fiables y funcionando de forma autónoma - para **Claude Code**, **OpenAI Codex** y el **Agents SDK**.

Los agentes de IA fallan de maneras predecibles. Ejecutan comandos destructivos, filtran secretos, se desvían de sus tareas, se quedan atascados en bucles o hacen push directamente a main. Si no se supervisan, los fallos pequeños se convierten en problemas mayores.

Failproof AI resuelve esto con **políticas** — reglas que se enganchan en cada llamada a herramientas del agente y pueden **detectar fallos**, **mitigarlos** (bloquear, instruir, sanear) y **alertarte** cuando algo requiere atención. Un panel de control local te permite revisar todo después.

Todo se ejecuta localmente. Ningún dato sale de tu máquina.

## Inicio rápido

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # activar políticas
failproofai                      # abrir el panel de control
```
