**Claude Code**、**OpenAI Codex**、**Agents SDK** に対応した、AIエージェントの信頼性を保ち自律的な実行を維持するフックとポリシーです。

AIエージェントは予測可能な形で失敗します。破壊的なコマンドを実行したり、シークレットを漏洩したり、タスクから逸脱したり、ループにはまったり、mainブランチに直接プッシュしたりします。放置すると、小さな失敗が連鎖的に拡大していきます。

Failproof AI はこの問題を **ポリシー** で解決します。ポリシーとは、エージェントのすべてのツール呼び出しにフックして、**失敗を検知**し、**対処**（ブロック、指示、サニタイズ）し、対応が必要な場合に**アラートを送る**ルールです。ローカルのダッシュボードで、後からすべての履歴を確認できます。

すべてはローカルで動作します。データが外部に送信されることはありません。

<Card title="26個の組み込みポリシー" icon="shield" href="/ja/built-in-policies">

</Card>

</Card>

## クイックスタート

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # ポリシーを有効化
failproofai                      # ダッシュボードを起動
```

完全なウォークスルーは[はじめ方](/ja/getting-started)ガイドをご覧ください。
