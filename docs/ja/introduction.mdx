**Claude Code**・**OpenAI Codex**・**Agents SDK** 向けに、AIエージェントを安定稼働・自律実行させるためのフックとポリシーを提供します。

AIエージェントは、予測可能な形で失敗します。破壊的なコマンドの実行、シークレットの漏洩、タスクからの逸脱、ループへのはまり込み、main ブランチへの直接プッシュ——放置すると、小さな失敗が連鎖的に拡大します。

Failproof AI はこれを**ポリシー**で解決します。ポリシーはエージェントのすべてのツール呼び出しにフックし、**失敗を検出**・**緩和**（ブロック、指示、サニタイズ）・**通知**を行います。ローカルダッシュボードで事後確認も可能です。

すべてローカルで動作します。データが外部に送信されることはありません。

## クイックスタート

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # ポリシーを有効化
failproofai                      # ダッシュボードを起動
```

詳しい手順は [Getting started](/ja/getting-started) ガイドをご覧ください。
