---

```bash
failproofai
```

`http://localhost:8020` पर वेब डैशबोर्ड शुरू करता है।

## विकल्प

| फ्लैग | विवरण |
|------|-------------|
| `--port <number>` | सुनने के लिए पोर्ट (डिफ़ॉल्ट: `8020`) |
| `--projects-path <path>` | Claude Code प्रोजेक्ट फ़ोल्डर कहां पाए जाते हैं, इसे ओवरराइड करें |
| `--allowed-origins <origins>` | विकास संसाधनों तक पहुंच के लिए अनुमत अल्पविराम-विभाजित होस्ट/IP |

## उदाहरण

```bash
# विभिन्न पोर्ट पर लॉन्च करें
failproofai --port 9000

# कस्टम प्रोजेक्ट पथ का उपयोग करें
failproofai --projects-path ~/my-claude-projects
```
