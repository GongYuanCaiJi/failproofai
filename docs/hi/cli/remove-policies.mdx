---
title: नीतियों को हटाएं
description: "Claude Code की सेटिंग्स से हुक प्रविष्टियों को हटाएं"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

Claude Code के `settings.json` से failproofai हुक प्रविष्टियों को हटाता है।

उपनाम: `failproofai p -u`

## विकल्प

| फ़्लैग | विवरण |
|------|-------------|
| `--scope user` | वैश्विक सेटिंग्स से हटाएं (डिफ़ॉल्ट) |
| `--scope project` | प्रोजेक्ट सेटिंग्स से हटाएं |
| `--scope local` | स्थानीय सेटिंग्स से हटाएं |
| `--scope all` | सभी स्कोप से एक साथ हटाएं |
| `--custom` / `-c` | कॉन्फ़िगरेशन से `customPoliciesPath` को साफ़ करें |

## व्यवहार

- **कोई नीति नाम नहीं** - सेटिंग्स फ़ाइल से सभी failproofai हुक प्रविष्टियों को हटाता है
- **विशिष्ट नाम** - उन नीतियों को अक्षम करता है लेकिन हुक को स्थापित रखता है

## उदाहरण

```bash
# सभी हुक को विश्व स्तर पर हटाएं
failproofai policies --uninstall

# एक विशिष्ट नीति को अक्षम करें (हुक को स्थापित रखता है)
failproofai policies --uninstall block-sudo

# प्रत्येक स्कोप से हुक हटाएं
failproofai policies --uninstall --scope all

# कस्टम नीतियों पथ को साफ़ करें
failproofai policies --uninstall --custom
```
