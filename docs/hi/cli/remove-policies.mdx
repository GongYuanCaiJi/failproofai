title: नीतियों को अनइंस्टॉल करें
description: "Claude Code की सेटिंग्स से हुक एंट्रीज़ को हटाएं"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

Claude Code के `settings.json` से failproofai हुक एंट्रीज़ को हटाता है।

उपनाम: `failproofai p -u`

## विकल्प

| फ़्लैग | विवरण |
|------|-------------|
| `--scope user` | ग्लोबल सेटिंग्स से हटाएं (डिफ़ॉल्ट) |
| `--scope project` | प्रोजेक्ट सेटिंग्स से हटाएं |
| `--scope local` | स्थानीय सेटिंग्स से हटाएं |
| `--scope all` | सभी स्कोप से एक साथ हटाएं |
| `--custom` / `-c` | कॉन्फ़िग से `customPoliciesPath` को साफ़ करें |

## व्यवहार

- **कोई नीति के नाम नहीं** - सेटिंग्स फ़ाइल से सभी failproofai हुक एंट्रीज़ को हटाता है
- **विशिष्ट नाम** - उन नीतियों को अक्षम करता है लेकिन हुक को इंस्टॉल रखता है

## उदाहरण

```bash
# सभी हुक को ग्लोबली हटाएं
failproofai policies --uninstall

# किसी विशिष्ट नीति को अक्षम करें (हुक इंस्टॉल रखता है)
failproofai policies --uninstall block-sudo

# हर स्कोप से हुक हटाएं
failproofai policies --uninstall --scope all

# कस्टम नीतियों पाथ को साफ़ करें
failproofai policies --uninstall --custom
```
