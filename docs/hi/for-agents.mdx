एक कमांड में अपने कोडिंग एजेंट को पूरा Failproof AI रेफरेंस दें। Claude Code, Cursor, Windsurf और किसी भी अन्य एजेंट के साथ काम करता है जो स्किल्स को सपोर्ट करता है।

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` पता लगाता है कि आपके पास कौन से एजेंट इंस्टॉल हैं और हर एक के लिए स्किल को सही फॉर्मेट में स्वचालित रूप से जोड़ता है।

## स्किल किन बातों को कवर करता है

| एरिया | क्या शामिल है |
|------|----------------|
| पॉलिसीज | बिल्ट-इन पॉलिसी नाम, इवेंट टाइप, पैरामीटर, सक्षम/अक्षम |
| कस्टम पॉलिसीज | `customPolicies.add()`, मैच फिल्टर, `allow`/`deny`/`instruct` API |
| कॉन्टेक्स्ट ऑब्जेक्ट | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| कॉन्फ़िगरेशन | `policies-config.json` संरचना, स्कोप मर्जिंग, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, स्कोप |
| डैशबोर्ड | सेशन व्यूअर, पॉलिसी एक्टिविटी, एनवायरनमेंट वेरिएबल्स |
| आर्किटेक्चर | हुक हैंडलर फ्लो, एक्जिट कोड, stdin/stdout कॉन्ट्रैक्ट |

## क्या स्किल पूरी है?

Mintlify नेविगेशन में सभी पेजों से `llms.txt` जेनरेट करता है। Failproof AI दस्तावेज़ पूरा API कवर करते हैं - हर पॉलिसी, विकल्प और उदाहरण शामिल है। अगर आपको कुछ मिसिंग लगता है, तो स्रोत `https://docs.befailproof.ai/llms-full.txt` पर है।

लक्षित संदर्भ के लिए, किसी विशिष्ट पेज से सीधे लिंक करें:

```bash
# सिर्फ कस्टम पॉलिसीज API
npx skills add https://docs.befailproof.ai/custom-policies

# सिर्फ बिल्ट-इन पॉलिसीज
npx skills add https://docs.befailproof.ai/built-in-policies
```
