हुकस और पॉलिसीस जो आपके AI एजेंटस को विश्वसनीय और स्वायत्त रूप से चलाए रखते हैं - **Claude Code**, **OpenAI Codex**, और **Agents SDK** के लिए।

AI एजेंटस पूर्वानुमानित तरीकों से विफल होते हैं। वे विनाशकारी आदेश चलाते हैं, गोपनीयता रिसाव करते हैं, काम से भटकते हैं, लूप में फंस जाते हैं, या सीधे मुख्य शाखा पर पुश करते हैं। बिना निरीक्षण के, छोटी विफलताएं बड़े मुद्दों में बदल जाती हैं।

Failproof AI इसे **पॉलिसीस** से हल करता है - नियम जो हर एजेंट टूल कॉल में हुक होते हैं और **विफलताओं का पता लगा सकते हैं**, **उन्हें कम कर सकते हैं** (ब्लॉक, निर्देश, सैनिटाइज), और **आपको सतर्क कर सकते हैं** जब किसी चीज को ध्यान देने की आवश्यकता हो। एक स्थानीय डैशबोर्ड आपको बाद में सब कुछ की समीक्षा करने देता है।

सब कुछ स्थानीय रूप से चलता है। कोई डेटा आपकी मशीन से बाहर नहीं जाता।

## त्वरित प्रारंभ

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # पॉलिसीस को सक्षम करें
failproofai                      # डैशबोर्ड लॉन्च करें
```
