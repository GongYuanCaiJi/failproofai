**AI फेलियर हैंडलिंग**, **एरर रिकवरी**, और **LLM रिलायबिलिटी** के लिए हुक्स और पॉलिसीज। अपने AI एजेंट्स को **Claude Code**, **OpenAI Codex**, **GitHub Copilot**, **Cursor Agent**, **OpenCode**, **Pi**, **Hermes**, **OpenClaw**, **Factory Droid**, **Devin CLI**, **Antigravity CLI**, और **Agents SDK** में विश्वसनीय और स्वायत्त रूप से चलाते रहें।

AI एजेंट्स प्रेडिक्टेबल तरीकों से फेल होते हैं। वे डिस्ट्रक्टिव कमांड्स चलाते हैं, सीक्रेट्स लीक करते हैं, टास्क से भटकते हैं, लूप्स में फंस जाते हैं, या सीधे मेन में पुश करते हैं। बिना निगरानी के, छोटी फेलियर्स बड़े आउटेजेस, लीक क्रेडेंशियल्स और खोए हुए काम में तब्दील हो जाती हैं।

FailproofAI इसे **पॉलिसीज** से हल करता है। ये नियम हर एजेंट टूल कॉल में हुक करते हैं ताकि **फेलियर्स को डिटेक्ट** किया जा सके, उन्हें **मिटिगेट** किया जा सके (ब्लॉक, इंस्ट्रक्ट, सैनिटाइज), और जब कुछ ध्यान की जरूरत हो तो **आपको अलर्ट** किया जा सके। एक लोकल डैशबोर्ड आपको हर टूल कॉल, एजेंट फेलियर, और रिकवरी एक्शन को बाद में रिव्यू करने देता है।

ट्रांसक्रिप्ट्स और पॉलिसी इवैलुएशन आपकी मशीन पर ही रहते हैं। डेटा सिर्फ तभी भेजा जाता है जब आप स्पष्ट रूप से कोई ऑनलाइन फीचर यूज करते हैं, जैसे ऑथेंटिकेटेड ऑडिट रिमाइंडर्स या इनवाइटेशन्स।

## त्वरित शुरुआत

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # पॉलिसीज को एनेबल करें (या स्किप करें — `failproofai` पहली रन पर सेटअप करने का ऑफर देगा)
failproofai                      # डैशबोर्ड लॉन्च करें
```
