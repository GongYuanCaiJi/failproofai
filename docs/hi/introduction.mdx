Hooks और policies जो आपके AI agents को विश्वसनीय रखते हैं और स्वायत्त रूप से चलाते हैं - **Claude Code**, **OpenAI Codex**, और **Agents SDK** के लिए।

AI agents पूर्वानुमानित तरीकों से विफल होते हैं। वे विनाशकारी कमांड चलाते हैं, गोपनीयता लीक करते हैं, कार्य से भटक जाते हैं, लूप में फंस जाते हैं, या सीधे main में push करते हैं। बिना देखभाल के, छोटी विफलताएं बड़ी समस्याओं में बदल जाती हैं।

Failproof AI इसे **policies** से हल करता है - नियम जो हर agent tool call में hook करते हैं और **विफलताओं का पता लगा सकते हैं**, **उन्हें कम कर सकते हैं** (block, instruct, sanitize), और **आपको सतर्क कर सकते हैं** जब कुछ ध्यान देने की आवश्यकता हो। एक local dashboard आपको बाद में सब कुछ देखने देता है।

सब कुछ स्थानीय रूप से चलता है। कोई डेटा आपकी मशीन से बाहर नहीं जाता।

## त्वरित शुरुआत

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # policies सक्षम करें
failproofai                      # dashboard लॉन्च करें
```
