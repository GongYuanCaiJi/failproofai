---
title: Xem các phiên làm việc
description: "Khởi chạy bảng điều khiển để duyệt các phiên hoạt động của agent và quản lý chính sách"
---

```bash
failproofai
```

Khởi chạy bảng điều khiển web tại `http://localhost:8020`.

## Tùy chọn

| Flag | Mô tả |
|------|-------------|
| `--port <number>` | Cổng để lắng nghe (mặc định: `8020`) |
| `--allowed-origins <origins>` | Các host/IP được phân tách bằng dấu phẩy được phép truy cập tài nguyên dev |

Để trỏ bảng điều khiển đến thư mục dự án Claude không phải mặc định, hãy đặt biến môi trường `CLAUDE_PROJECTS_PATH` khi khởi chạy.

## Ví dụ

```bash
# Launch on a different port
failproofai --port 9000

# Use a custom Claude projects path via environment variable
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
