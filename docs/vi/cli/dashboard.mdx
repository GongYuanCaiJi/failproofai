---
title: Xem các phiên làm việc
description: "Khởi động dashboard để duyệt các phiên làm việc của agent và quản lý các chính sách"
---

```bash
failproofai
```

Khởi động dashboard web tại `http://localhost:8020`.

## Tùy chọn

| Flag | Mô tả |
|------|-------|
| `--port <number>` | Cổng để lắng nghe (mặc định: `8020`) |
| `--allowed-origins <origins>` | Các host/IP được phép truy cập tài nguyên dev, cách nhau bằng dấu phẩy |

Để trỏ dashboard tới thư mục dự án Claude không phải mặc định, hãy đặt biến môi trường `CLAUDE_PROJECTS_PATH` khi khởi động.

## Ví dụ

```bash
# Khởi động trên một cổng khác
failproofai --port 9000

# Sử dụng đường dẫn dự án Claude tùy chỉnh thông qua biến môi trường
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
