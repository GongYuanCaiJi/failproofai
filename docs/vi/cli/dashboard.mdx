---
title: Xem các phiên làm việc
description: "Khởi chạy bảng điều khiển để duyệt các phiên tác nhân và quản lý chính sách"
---

```bash
failproofai
```

Khởi chạy bảng điều khiển web tại `http://localhost:8020`.

## Các tùy chọn

| Cờ | Mô tả |
|------|-------------|
| `--port <number>` | Cổng để lắng nghe (mặc định: `8020`) |
| `--allowed-origins <origins>` | Các máy chủ/IP được phân tách bằng dấu phẩy được phép truy cập tài nguyên phát triển |

Để trỏ bảng điều khiển tới thư mục dự án Claude không mặc định, hãy đặt biến môi trường `CLAUDE_PROJECTS_PATH` khi khởi chạy.

## Ví dụ

```bash
# Khởi chạy trên một cổng khác
failproofai --port 9000

# Sử dụng đường dẫn dự án Claude tùy chỉnh thông qua biến môi trường
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
