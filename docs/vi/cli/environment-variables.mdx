---

## Dashboard

| Biến | Mô tả |
|----------|-------------|
| `PORT` | Cổng dashboard (mặc định: `8020`) |
| `CLAUDE_PROJECTS_PATH` | Ghi đè vị trí tìm kiếm thư mục dự án Claude Code |
| `FAILPROOFAI_DISABLE_PAGES=policies,projects` | Các trang dashboard được ẩn, phân cách bằng dấu phẩy |
| `FAILPROOFAI_ALLOWED_DEV_ORIGINS` | Các host/IP được phép truy cập tài nguyên dev. Giống như `--allowed-origins`. |

## Logging

| Biến | Mô tả |
|----------|-------------|
| `FAILPROOFAI_LOG_LEVEL=info\|warn\|error` | Mức log của server (mặc định: `warn`) |
| `FAILPROOFAI_HOOK_LOG_FILE` | Đường dẫn tệp log hook tùy chỉnh, hoặc `true` cho mặc định (`~/.failproofai/logs/hooks.log`) |

## Telemetry

| Biến | Mô tả |
|----------|-------------|
| `FAILPROOFAI_TELEMETRY_DISABLED=1` | Vô hiệu hóa telemetry sử dụng nặc danh |

## LLM (để đánh giá chính sách)

| Biến | Mô tả |
|----------|-------------|
| `FAILPROOFAI_LLM_BASE_URL` | Điểm cuối LLM API (mặc định: `https://api.openai.com/v1`) |
| `FAILPROOFAI_LLM_API_KEY` | Khóa API cho các chính sách được hỗ trợ bởi LLM |
| `FAILPROOFAI_LLM_MODEL` | Tên mô hình (mặc định: `gpt-4o-mini`) |
