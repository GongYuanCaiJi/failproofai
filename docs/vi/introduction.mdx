Hooks và policies giúp các AI agents của bạn hoạt động đáng tin cậy và tự động - cho **Claude Code**, **OpenAI Codex**, và **Agents SDK**.

Các AI agents thất bại theo những cách dễ dự đoán. Chúng chạy các lệnh có tính phá hủy, để lộ bí mật, lệch khỏi nhiệm vụ, bị kẹt trong vòng lặp, hoặc push trực tiếp lên main. Nếu không được giám sát, những lỗi nhỏ sẽ dẫn đến hậu quả lớn.

Failproof AI giải quyết vấn đề này bằng **policies** - các quy tắc hook vào mỗi tool call của agent và có thể **phát hiện lỗi**, **giảm thiểu chúng** (block, instruct, sanitize), và **cảnh báo cho bạn** khi cần chú ý. Một bảng điều khiển cục bộ cho phép bạn xem lại mọi thứ sau.

Tất cả chạy cục bộ. Không có dữ liệu nào rời khỏi máy của bạn.

## Khởi động nhanh

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```

Xem hướng dẫn [Getting started](/vi/getting-started) để biết hướng dẫn đầy đủ.
