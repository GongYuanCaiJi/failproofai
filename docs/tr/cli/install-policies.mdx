---
title: Politikaları yükle
description: "Politikaları etkinleştirin ve her ajan araç çağrısında çalışmasını sağlayın"
---

```bash
failproofai policies --install [policy-names...] [options]
```

failproofai'nin araç çağrılarını engelleyebilmesi için yüklü ajan CLI'nizin (Claude Code, OpenAI Codex veya GitHub Copilot CLI _(beta)_) ayarlar dosyasına kanca girişleri yazar.

Takma adlar: `failproofai p -i`

## Seçenekler

| Bayrak | Açıklama |
|------|-------------|
| `--cli claude\|codex\|copilot` | Kurulacak Ajan CLI'leri; boşlukla ayrılmış (örn. `--cli claude codex copilot`) veya tekrar edilmiş. Yüklü CLI'leri tespit etmek ve sormak için belirtmeyin. |
| `--scope user` | Kullanıcı kapsamı ayarlar dosyasına yükleyin (Claude: `~/.claude/settings.json`; Codex: `~/.codex/hooks.json`; Copilot: `~/.copilot/hooks/failproofai.json`). Varsayılan. |
| `--scope project` | Proje kapsamı ayarlar dosyasına yükleyin (Claude: `<cwd>/.claude/settings.json`; Codex: `<cwd>/.codex/hooks.json`; Copilot: `<cwd>/.github/hooks/failproofai.json`). |
| `--scope local` | Yalnızca Claude — `<cwd>/.claude/settings.local.json` dosyasına yükleyin. Codex ve Copilot'un `local` kapsamı yoktur. |
| `--custom <path>` / `-c` | Özel kanca politikalarını içeren JS dosyasının yolu |

## Davranış

- **Politika adı yok** - politika seçmek için etkileşimli istemi açar
- **Belirli adlar** - bu politikaları etkinleştirir (zaten etkinleştirilenler korunur)
- **`all`** - tüm kullanılabilir politikaları etkinleştirir

Kurulum eklemeli: `--install` komutunu tekrar çalıştırmak yeni politikaları ekler ancak mevcut olanları kaldırmaz.

## Örnekler

```bash
# Tüm varsayılan politikaları genel olarak yükle (etkileşimli)
failproofai policies --install

# Geçerli proje için belirli politikaları yükle
failproofai policies --install block-sudo sanitize-api-keys --scope project

# Tüm politikaları aynı anda etkinleştir
failproofai policies --install all

# Özel politikalar dosyası ile yükle
failproofai policies --install --custom ./my-policies.js

# OpenAI Codex için yükle (proje kapsamı)
failproofai policies --install --cli codex --scope project

# GitHub Copilot CLI (beta) için geçerli proje içinde yükle
failproofai policies --install --cli copilot --scope project

# Üç CLI için aynı anda yükle
failproofai policies --install --cli claude codex copilot
```

`--custom <path>` sağlandığında, dosya hemen doğrulanır - en az bir kez `customPolicies.add()` çağrısı yapmalıdır. Çözümlenen yol `policies-config.json` dosyasına `customPoliciesPath` olarak kaydedilir.
