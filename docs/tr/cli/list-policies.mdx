---
title: Politikaları Listele
description: "Hangi politikaların etkinleştirildiğini, parametrelerini ve özel politikaları göreceğiniz"
---

```bash
failproofai policies
```

Tüm politikaları durum, yapılandırılmış parametreler ve özel politikalar ile gösterir.

## Örnek çıktı

```text
Failproof AI Hook Policies (user)

  Status  Name                          Description
  ──────  ──────────────────────────────────────────────────────────────
  ✓       block-sudo                    Sudo komutlarını engelle
            allowPatterns: ["sudo systemctl status"]
  ✓       block-rm-rf                   Özyinelemeli silmeleri engelle
  ✗       block-curl-pipe-sh            curl|bash kalıplarını engelle
  ✓       sanitize-api-keys             API anahtarlarını çıktıdan gizle
            additionalPatterns: [{ regex: "MY_TOKEN_...", label: "..." }]

  ── Custom Policies (/home/alice/myproject/my-policies.js) ──────────────
  ✓       require-jira-ticket           Bilet olmayan commitleri engelle
  ✓       approval-gate                 Yıkıcı işlemler için onay kapısı
```

`policyParams` içindeki bilinmeyen anahtarlar burada işaretlenir, böylece yazım hatalarını erken fark edebilirsiniz.
