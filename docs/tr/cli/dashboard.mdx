---
title: Oturumları görüntüle
description: "Aracı oturumlarına göz atmak ve politikaları yönetmek için panoyu başlatın"
---

```bash
failproofai
```

Web panosunu `http://localhost:8020` adresinde başlatır.

## Seçenekler

| Bayrak | Açıklama |
|------|-------------|
| `--port <number>` | Dinlenecek port (varsayılan: `8020`) |
| `--allowed-origins <origins>` | Geliştirici kaynaklarına erişim için izin verilen virgülle ayrılmış ana bilgisayarlar/IP'ler |

Panoyu varsayılan olmayan bir Claude proje klasörüne yönlendirmek için, başlatırken `CLAUDE_PROJECTS_PATH` ortam değişkenini ayarlayın.

## Örnekler

```bash
# Farklı bir portta başlatma
failproofai --port 9000

# Ortam değişkeni aracılığıyla özel bir Claude projeleri yolu kullanma
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
