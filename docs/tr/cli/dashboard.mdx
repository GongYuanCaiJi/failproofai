---
title: Oturumları Görüntüle
description: "Aracı oturumlarına göz atmak ve ilkeleri yönetmek için panoyu başlatın"
---

```bash
failproofai
```

Web panosunu `http://localhost:8020` adresinde başlatır.

## Seçenekler

| Bayrak | Açıklama |
|------|-------------|
| `--port <number>` | Dinlenecek bağlantı noktası (varsayılan: `8020`) |
| `--projects-path <path>` | Claude Code proje klasörlerinin bulunduğu konumu geçersiz kıl |
| `--allowed-origins <origins>` | Geliştirme kaynaklarına erişmesine izin verilen virgülle ayrılmış ana bilgisayarlar/IP'ler |

## Örnekler

```bash
# Farklı bir bağlantı noktasında başlat
failproofai --port 9000

# Özel bir proje yolu kullan
failproofai --projects-path ~/my-claude-projects
```
