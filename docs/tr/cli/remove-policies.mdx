---
title: Politikaları kaldır
description: "Claude Code'un ayarlarından hook girdilerini kaldır"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

failproofai hook girdilerini Claude Code'un `settings.json` dosyasından kaldırır.

Takma adlar: `failproofai p -u`

## Seçenekler

| Bayrak | Açıklama |
|------|-------------|
| `--scope user` | Global ayarlardan kaldır (varsayılan) |
| `--scope project` | Proje ayarlarından kaldır |
| `--scope local` | Yerel ayarlardan kaldır |
| `--scope all` | Tüm kapsamlardan aynı anda kaldır |
| `--custom` / `-c` | `customPoliciesPath` yapılandırmasını temizle |

## Davranış

- **Politika adı yok** - ayarlar dosyasından tüm failproofai hook girdilerini kaldırır
- **Belirli adlar** - bu politikaları devre dışı bırakır ancak hook'ları yüklü tutar

## Örnekler

```bash
# Tüm hook'ları global olarak kaldır
failproofai policies --uninstall

# Belirli bir politikayı devre dışı bırak (hook'ları yüklü tutar)
failproofai policies --uninstall block-sudo

# Hook'ları her kapsamdan kaldır
failproofai policies --uninstall --scope all

# Özel politikalar yolunu temizle
failproofai policies --uninstall --custom
```
