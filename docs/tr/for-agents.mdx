Failproof AI referansının tamamını bir komutla kodlama ajanınıza ekleyin. Claude Code, Cursor, Windsurf ve becerileri destekleyen diğer tüm ajanlarla çalışır.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` hangi ajanların yüklenmiş olduğunu algılar ve beceriyi her biri için otomatik olarak doğru formatta ekler.

## Becerinin kapsamı

| Alan | Ne dahildir |
|------|------------|
| İlkeler | Yerleşik ilke adları, olay türleri, parametreler, etkinleştir/devre dışı bırak |
| Özel ilkeler | `customPolicies.add()`, eşleştirme filtreleri, `allow`/`deny`/`instruct` API |
| Context nesnesi | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| Yapılandırma | `policies-config.json` yapısı, kapsam birleştirme, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, kapsamlar |
| Pano | Oturum görüntüleyici, ilke etkinliği, ortam değişkenleri |
| Mimari | Hook işleyicisi akışı, çıkış kodları, stdin/stdout sözleşmesi |

## Beceri tam mı?

Mintlify, `llms.txt` dosyasını gezintideki tüm sayfalardan oluşturur. Failproof AI dokümanları tam API'yi kapsar - her ilke, seçenek ve örnek dahildir. Eksik bir şey bulursanız, kaynak `https://docs.befailproof.ai/llms-full.txt` adresindedir.

Hedefli içerik için doğrudan belirli bir sayfaya bağlantı verin:

```bash
# Yalnızca özel ilkeler API'si
npx skills add https://docs.befailproof.ai/custom-policies

# Yalnızca yerleşik ilkeler
npx skills add https://docs.befailproof.ai/built-in-policies
```
