Failproof AI referansının tamamını kodlama ajanınıza tek bir komutla ekleyin. Claude Code, Cursor, Windsurf ve becerileri destekleyen diğer tüm ajanlarla çalışır.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` hangi ajanların yüklü olduğunu algılar ve beceriyi her biri için otomatik olarak doğru formatta ekler.

## Beceri neleri kapsar

| Alan | İçerik |
|------|--------|
| Politikalar | Yerleşik politika adları, olay türleri, parametreler, etkinleştir/devre dışı bırak |
| Özel politikalar | `customPolicies.add()`, eşleşme filtreleri, `allow`/`deny`/`instruct` API |
| Bağlam nesnesi | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| Yapılandırma | `policies-config.json` yapısı, kapsam birleştirme, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, kapsamlar |
| Dashboard | Oturum görüntüleyici, politika aktivitesi, ortam değişkenleri |
| Mimari | Hook işleyici akışı, çıkış kodları, stdin/stdout sözleşmesi |

## Beceri tam mı?

Mintlify, navigasyondaki tüm sayfalardan `llms.txt` oluşturur. Failproof AI belgelerinde tam API kapsanmaktadır - her politika, seçenek ve örnek dahildir. Eksik bir şey bulursanız, kaynak `https://docs.befailproof.ai/llms-full.txt` adresindedir.

Hedefli bağlam için doğrudan belirli bir sayfaya bağlantı verin:

```bash
# Yalnızca özel politikalar API'si
npx skills add https://docs.befailproof.ai/custom-policies

# Yalnızca yerleşik politikalar
npx skills add https://docs.befailproof.ai/built-in-policies
```
