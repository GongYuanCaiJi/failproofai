AI aracılarınızı güvenilir ve otonom olarak çalışır tutacak hook'lar ve politikalar - **Claude Code**, **OpenAI Codex** ve **Agents SDK** için.

AI aracıları tahmin edilebilir şekillerde başarısız olur. Yıkıcı komutlar çalıştırırlar, sırları sızdırırlar, görevden saparlar, döngülere takılırlar ya da doğrudan ana branch'e push yaparlar. Gözetimsiz bırakıldığında, küçük hatalar basamaklanır.

Failproof AI bunu **politikalar** ile çözer - her agent araç çağrısına hook'lanan ve **hataları tespit edebilen**, **onları azaltabilen** (engelle, talimat ver, temizle) ve **dikkat gerektiğinde sizi uyarabilen** kurallar. Yerel bir dashboard tüm işlemleri incelemenizi sağlar.

Her şey yerel olarak çalışır. Hiçbir veri makinenizden çıkmaz.

## Hızlı başlangıç

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # politikaları etkinleştir
failproofai                      # dashboard'ı başlat
```
