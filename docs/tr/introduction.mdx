AI aracılarınızı güvenilir tutun ve otonom olarak çalışmalarını sağlayan hook'lar ve politikalar - **Claude Code**, **OpenAI Codex** ve **Agents SDK** için.

AI aracıları öngörülebilir şekillerde başarısız olur. Yıkıcı komutlar çalıştırırlar, gizli anahtarları sızdırırlar, görevden sapıp kalırlar, döngülerde takılırlar veya doğrudan main branch'ine push yaparlar. Kontrol edilmediğinde, küçük arızalar büyür.

Failproof AI bunu **politikalar** ile çözer - her bir agent araç çağrısına bağlanabilen ve **hataları tespit edebilen**, **bunları azaltabilen** (engelleme, yönetim, temizleme) ve **dikkat gereken durumları** size **bildirebilen** kurallar. Yerel bir dashboard sayesinde daha sonra her şeyi gözden geçirebilirsiniz.

Her şey yerel olarak çalışır. Hiç veri makinenizden çıkmaz.

## Hızlı başlangıç

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # politikaları etkinleştir
failproofai                      # dashboard'u başlat
```
