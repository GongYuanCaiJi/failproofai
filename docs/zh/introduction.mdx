让 AI 智能体保持可靠、自主运行的钩子和策略——适用于 **Claude Code**、**OpenAI Codex** 以及 **Agents SDK**。

AI 智能体的失败方式往往是可以预见的：执行破坏性命令、泄露密钥、偏离任务、陷入死循环，或者直接推送到主分支。一旦无人看管，小问题便会接连引发更大的故障。

Failproof AI 通过**策略**来解决这些问题——策略是一套规则，能够钩入智能体的每一次工具调用，**检测故障**、**主动缓解**（阻止、指示、脱敏），并在需要人工介入时**发出提醒**。本地仪表盘让你随时复盘所有操作记录。

所有功能均在本地运行，数据不会离开你的设备。

## 快速开始

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # 启用策略
failproofai                      # 启动仪表盘
```

完整操作流程请参阅[入门指南](/zh/getting-started)。
