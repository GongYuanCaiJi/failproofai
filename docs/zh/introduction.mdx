为 **Claude Code**、**OpenAI Codex** 和 **Agents SDK** 提供钩子和策略，让您的 AI 智能体保持可靠并自主运行。

AI 智能体的失败方式往往是可以预见的：执行破坏性命令、泄露密钥、偏离任务、陷入循环，或直接推送到主分支。一旦无人监管，小问题就会引发连锁故障。

Failproof AI 通过**策略**来解决这些问题——这些规则挂钩到每一次智能体工具调用中，能够**检测故障**、**进行缓解**（拦截、指导、清理），并在需要关注时**及时提醒您**。本地仪表板让您事后可以全面复盘。

所有处理均在本地完成，数据不会离开您的机器。

## 快速开始

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # 启用策略
failproofai                      # 启动仪表板
```

完整操作指南请参阅[入门指南](/zh/getting-started)。
