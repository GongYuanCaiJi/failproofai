观测界面用于实时查看 Agent 的运行状态，并深入分析任意单次运行的详情。这里的所有内容均为实时数据，按组织维度归类，支持按日期范围、环境、Agent 和会话进行筛选，让你能在数秒内从"感觉有些不对劲"定位到具体的运行记录。

![实时事件流，按类型进行颜色区分，支持按环境、Agent 和会话筛选](/agenteye/images/events-stream.png)

共有四个子界面，每个均有独立页面：

- **[事件流](/zh/agenteye/event-stream)**：所有 Agent 每次运行的实时逐步记录，最新内容排在最前。这是你的组织主页，也是排查问题的第一站。
- **[会话与执行图](/zh/agenteye/sessions)**：将事件汇总为每次运行一行的视图，并以类 Git 的图形展示每次运行的完整过程。
- **[性能指标](/zh/agenteye/telemetry)**：针对模型、工具和 Hook 的延迟热图及 p50/p95/p99 关键指标，让尾部延迟峰值与中位数之间的差异一目了然。
- **[错误追踪](/zh/agenteye/error-tracking)**：将所有异常统一汇聚到一个排查界面，一键从触发的告警跳转到对应的问题运行记录。
