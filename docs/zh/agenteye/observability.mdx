观测界面用于实时查看代理正在执行的操作，并深入分析任意单次运行。这里的所有内容均为实时数据，以您的组织为范围，支持按日期范围、环境、代理和会话进行筛选，让您能够在数秒内从"感觉有些不对"定位到具体的运行记录。

![实时事件流，按类型进行颜色编码，支持按环境、代理和会话筛选](/agenteye/images/events-stream.png)

共四个界面，每个均有独立页面：

- **[事件流](/zh/agenteye/event-stream)**：所有代理每次运行的实时逐步记录，最新条目优先显示。这是您组织的主页，也是问题分类排查的第一站。
- **[会话与执行图](/zh/agenteye/sessions)**：将事件汇总为每次运行一行的视图，并以类似 git 的图示展示每次运行的展开过程。
- **[性能指标](/zh/agenteye/telemetry)**：模型、工具和 hook 的延迟热力图及 p50/p95/p99 关键指标，让尾部延迟峰值从中位数中一目了然地凸显出来。
- **[错误追踪](/zh/agenteye/error-tracking)**：一个统一的错误分类排查界面，汇集所有异常情况，一键从触发的告警跳转至引发问题的运行记录。
