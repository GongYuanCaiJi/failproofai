Hooks ופוליסיות השומרות על אמינות הסוכנים שלך והפעלתם באופן עצמאי - עבור **Claude Code**, **OpenAI Codex**, ו-**Agents SDK**.

סוכנים בינה מלאכותית נכשלים בדרכים צפויות. הם מריצים פקודות הרסניות, דוליפים סודות, סוטים מהמשימה המקורית, תקועים בלולאות, או דוחפים ישירות ל-main. אם מושאירים ללא השגחה, כשלים קטנים גדלים למפולת.

Failproof AI פותר זאת באמצעות **פוליסיות** - כללים המתחברים לכל קריאת tool של הסוכן ויכולים **לגלות כשלים**, **להקל עליהם** (חסימה, הנחיה, ניקוי), ו**להתריע אותך** כשמשהו דורש תשומת לב. לוח בקרה מקומי מאפשר לך לבדוק הכל לאחר מכן.

הכל פועל באופן מקומי. אף נתון לא משאיר את המכונה שלך.

## התחל בדקה אחת

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # הפעל פוליסיות
failproofai                      # הפעל את לוח הבקרה
```
