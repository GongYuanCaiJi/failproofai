Hooks ופוליסיות השומרות על אמינות סוכנים AI שלך והנעה אוטונומית - עבור **Claude Code**, **OpenAI Codex**, וה-**Agents SDK**.

סוכנים AI נכשלים בדרכים ניתנות לחיזוי. הם מריצים פקודות הרסניות, דולפים סודות, סוטים מהמשימה, תקועים בלולאות, או דוחפים ישירות ל-main. כשמעזבים בלי השגחה, כשלים קטנים מתגברים.

Failproof AI פותר זאת עם **פוליסיות** - כללים שתופסים כל קריאת כלי של סוכן ויכולים **לגלות כשלים**, **להפחית אותם** (חסימה, הוראה, ניקוי), ו**להתריע אותך** כשמשהו זקוק לתשומת לב. דשבורד מקומי מאפשר לך לבדוק הכל אחר כך.

הכל פועל מקומית. אף נתון לא עוזב את המכונה שלך.

## התחלה מהירה

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```
