[![npm weekly downloads](https://img.shields.io/npm/dw/failproofai?style=flat-square&color=2ea44f)](https://www.npmjs.com/package/failproofai)

ווסדקים ומדיניויות ל**טיפול בכישלונות AI**, **התאוששות משגיאות**, ו**אמינות LLM**. שמור על סוכני AI שלך אמינים וקטנו בעצמאות על פני **Claude Code**, **OpenAI Codex**, **GitHub Copilot**, **Cursor Agent**, **OpenCode**, **Pi**, **Hermes**, **OpenClaw**, **Factory Droid**, **Devin CLI**, **Antigravity CLI**, וה**Agents SDK**.

סוכני AI נכשלים בדרכים צפויות. הם מריצים פקודות הרסניות, דולפים סודיות, סוטים מהמשימה, נתקעים בלולאות, או דוחפים ישירות ל-main. ללא השגחה, כישלונות קטנים הופכים להפסקות, דליפות בפרטי גישה, והעבודה שאבדה.

FailproofAI פותר זאת עם **מדיניויות**. כללים אלה תופסים בכל קריאת כלי סוכן כדי **לגלות כישלונות**, **להקל עליהם** (חסימה, הוראה, עיקום), ו**להודיע לך** כאשר משהו זקוק לתשומת לב. לוח מחוונים מקומי מאפשר לך לבדוק כל קריאת כלי, כישל סוכן, ופעולת התאוששות בעתיד.

תמלילים והערכת מדיניות נשארים במחשב שלך. נתונים נשלחים רק כאשר אתה משתמש במפורש בתכונה מקוונת, כגון תזכורות ביקורת מאומתות או הזמנות.

## התחלה מהירה

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # הפעל מדיניויות (או דלג — `failproofai` יציע להגדיר אותן בהפעלה הראשונה)
failproofai                      # הפעל את לוח המחוונים
```
