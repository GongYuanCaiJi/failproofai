---
title: צפייה בהפעלות
description: "הפעל את לוח הבקרה כדי לעיין בהפעלות של סוכנים וניהול מדיניות"
---

```bash
failproofai
```

מפעיל את לוח הבקרה בכתובת `http://localhost:8020`.

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--port <number>` | היציאה שיש להאזין אליה (ברירת מחדל: `8020`) |
| `--projects-path <path>` | דרוס את המיקום בו נמצאים תיקיות פרויקט Claude Code |
| `--allowed-origins <origins>` | רשימה מופרדת בפסיקים של מארחים/כתובות IP המורשות לגשת למשאבי dev |

## דוגמאות

```bash
# הפעל ביציאה שונה
failproofai --port 9000

# השתמש בנתיב פרויקטים מותאם אישית
failproofai --projects-path ~/my-claude-projects
```
