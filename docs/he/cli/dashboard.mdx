---
title: צפייה בהפעילויות
description: "הפעל את לוח הבקרה לעיון בהפעילויות של סוכן וניהול מדיניות"
---

```bash
failproofai
```

מפעיל את לוח הבקרה של הרשת בכתובת `http://localhost:8020`.

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--port <number>` | היציאה להאזנה (ברירת מחדל: `8020`) |
| `--allowed-origins <origins>` | מארחים/כתובות IP מופרדות בפסיקים המורשות לגשת למשאבי פיתוח |

כדי להצביע את לוח הבקרה לתיקיית פרויקט Claude שאינה ברירת מחדל, הגדר את משתנה הסביבה `CLAUDE_PROJECTS_PATH` בעת ההפעלה.

## דוגמאות

```bash
# הפעל על יציאה שונה
failproofai --port 9000

# השתמש בנתיב פרויקטים Claude מותאם אישית דרך משתנה סביבה
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
