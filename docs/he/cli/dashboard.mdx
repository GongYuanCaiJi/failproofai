---
title: צפייה בסשנים
description: "הפעל את לוח הבקרה לעיון בסשני סוכנים וניהול מדיניות"
---

```bash
failproofai
```

מפעיל את לוח הבקרה באינטרנט בכתובת `http://localhost:8020`.

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--port <number>` | הפורט להאזנה (ברירת מחדל: `8020`) |
| `--allowed-origins <origins>` | רשימה מופרדת בפסיקים של שרתים/כתובות IP המורשים לגשת למשאבי הפיתוח |

כדי להצביע את לוח הבקרה בתיקייה של פרויקט Claude שאינה ברירת המחדל, הגדר את משתנה הסביבה `CLAUDE_PROJECTS_PATH` בעת ההפעלה.

## דוגמאות

```bash
# הפעל בפורט שונה
failproofai --port 9000

# השתמש בנתיב פרויקטים Claude מותאם אישית דרך משתנה סביבה
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
