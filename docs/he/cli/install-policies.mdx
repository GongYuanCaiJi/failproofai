---
title: התקנת מדיניות
description: "אפשרו מדיניות כך שהן יפעלו בכל קריאת כלי סוכן"
---

```bash
failproofai policies --install [policy-names...] [options]
```

כותב ערכי hook לתוך קובץ ההגדרות של ה-CLI של הסוכן המותקן שלכם (Claude Code, OpenAI Codex, או GitHub Copilot CLI _(beta)_) כך ש-failproofai יעצור קריאות כלים.

כינויים: `failproofai p -i`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--cli claude\|codex\|copilot` | ה-CLI(s) של הסוכן להתקנה; מופרדים בחלל (לדוגמה `--cli claude codex copilot`) או חוזרים. השמיטו כדי לגלות CLIs מותקנים ולקבל הנחיות. |
| `--scope user` | התקן לקובץ הגדרות בהיקף משתמש (Claude: `~/.claude/settings.json`; Codex: `~/.codex/hooks.json`; Copilot: `~/.copilot/hooks/failproofai.json`). ברירת המחדל. |
| `--scope project` | התקן לקובץ הגדרות בהיקף פרויקט (Claude: `<cwd>/.claude/settings.json`; Codex: `<cwd>/.codex/hooks.json`; Copilot: `<cwd>/.github/hooks/failproofai.json`). |
| `--scope local` | Claude בלבד — מתקין לתוך `<cwd>/.claude/settings.local.json`. Codex ו-Copilot אין להם היקף `local`. |
| `--custom <path>` / `-c` | נתיב לקובץ JS המכיל מדיניות hook מותאמת אישית |

## התנהגות

- **ללא שמות מדיניות** - פותח הנחיה אינטראקטיבית לבחירת מדיניות
- **שמות ספציפיים** - מאפשרים את אותן מדיניות (מוסיפים לכל כבר מופעלות)
- **`all`** - מאפשרים כל מדיניות זמינה

התקנה היא תוסף: הפעלת `--install` שוב מוסיפה מדיניות חדשה ללא הסרת מדיניות קיימות.

## דוגמאות

```bash
# התקן את כל המדיניות הברירות המחדל ברחבי העולם (אינטראקטיבי)
failproofai policies --install

# התקן מדיניות ספציפיות לפרויקט הנוכחי
failproofai policies --install block-sudo sanitize-api-keys --scope project

# אפשר את כל המדיניות בבת אחת
failproofai policies --install all

# התקן עם קובץ מדיניות מותאם אישית
failproofai policies --install --custom ./my-policies.js

# התקן עבור OpenAI Codex (היקף פרויקט)
failproofai policies --install --cli codex --scope project

# התקן עבור GitHub Copilot CLI (beta) עבור הפרויקט הנוכחי
failproofai policies --install --cli copilot --scope project

# התקן לכל שלושת ה-CLIs בו-זמנית
failproofai policies --install --cli claude codex copilot
```

כאשר `--custom <path>` סופק, הקובץ מאומת מיד - הוא חייב להתקשר ל-`customPolicies.add()` לפחות פעם אחת. הנתיב המפותל נשמר ב-`policies-config.json` כ-`customPoliciesPath`.
