title: הסרת מדיניות
description: "הסרת ערכי hook מהגדרות Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

מסיר ערכי failproofai hook מהקובץ `settings.json` של Claude Code.

כינויים: `failproofai p -u`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--scope user` | הסרה מההגדרות הגלובליות (ברירת מחדל) |
| `--scope project` | הסרה מהגדרות הפרויקט |
| `--scope local` | הסרה מהגדרות המקומיות |
| `--scope all` | הסרה מכל ההיקפים בו-זמנית |
| `--custom` / `-c` | ניקוי ה-`customPoliciesPath` מהתצורה |

## התנהגות

- **ללא שמות מדיניות** - מסיר את כל ערכי failproofai hook מקובץ ההגדרות
- **שמות ספציפיים** - מנטרל את המדיניויות הללו אך משאיר את ה-hooks מותקנים

## דוגמאות

```bash
# הסרת כל ה-hooks בצורה גלובלית
failproofai policies --uninstall

# השבתת מדיניות ספציפית (שמר את ה-hooks מותקנים)
failproofai policies --uninstall block-sudo

# הסרת hooks מכל היקף
failproofai policies --uninstall --scope all

# ניקוי נתיב המדיניות המותאמות
failproofai policies --uninstall --custom
```
