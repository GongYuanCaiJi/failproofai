---
title: הסרת מדיניות
description: "הסרת רשומות hook מהגדרות Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

מסירה רשומות failproofai hook מ-`settings.json` של Claude Code.

כינויים: `failproofai p -u`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--scope user` | הסרה מהגדרות גלובליות (ברירת מחדל) |
| `--scope project` | הסרה מהגדרות הפרוייקט |
| `--scope local` | הסרה מהגדרות מקומיות |
| `--scope all` | הסרה מכל ההיקפים בו-זמנית |
| `--custom` / `-c` | ניקוי של `customPoliciesPath` מהקונפיגורציה |

## התנהגות

- **ללא שמות מדיניות** - מסירה את כל רשומות failproofai hook מקובץ ההגדרות
- **שמות ספציפיים** - משביתה מדיניות אלה אך שומרת את ה-hooks מותקנות

## דוגמאות

```bash
# הסרת כל ה-hooks ברמה גלובלית
failproofai policies --uninstall

# השבתת מדיניות ספציפית (שומרת ה-hooks מותקנות)
failproofai policies --uninstall block-sudo

# הסרת hooks מכל היקף
failproofai policies --uninstall --scope all

# ניקוי נתיב המדיניות המותאמות
failproofai policies --uninstall --custom
```
