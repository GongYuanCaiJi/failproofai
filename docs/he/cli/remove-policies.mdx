---
title: הסרת policies
description: "הסרת hook entries מהגדרות Claude Code"
---

```bash
failproofai policies --uninstall [policy-names...] [options]
```

מסירה failproofai hook entries מ-`settings.json` של Claude Code.

כינויים: `failproofai p -u`

## אפשרויות

| דגל | תיאור |
|------|-------------|
| `--scope user` | הסרה מהגדרות גלובליות (ברירת מחדל) |
| `--scope project` | הסרה מהגדרות פרויקט |
| `--scope local` | הסרה מהגדרות מקומיות |
| `--scope all` | הסרה מכל ההיקפים בו-זמנית |
| `--custom` / `-c` | ניקוי `customPoliciesPath` מהתצורה |

## התנהגות

- **ללא שמות policies** - מסירה את כל failproofai hook entries מקובץ ההגדרות
- **שמות ספציפיים** - מכבה את ה-policies הללו אך משאירה hooks מותקנים

## דוגמאות

```bash
# הסרת כל ה-hooks בעולם
failproofai policies --uninstall

# השבתת policy ספציפי (משאירה hooks מותקנים)
failproofai policies --uninstall block-sudo

# הסרת hooks מכל היקף
failproofai policies --uninstall --scope all

# ניקוי נתיב custom policies
failproofai policies --uninstall --custom
```
