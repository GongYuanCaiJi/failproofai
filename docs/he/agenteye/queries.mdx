שאל כל שאלה על נתוני הסוכן שלך וקבל תשובה בשניות. FailproofAI Observability נותן לך ספריית שאילתות שמורות וערוכות להרצה על האירועים וההערכות שלך, כדי שתתחיל מדוגמה עובדת ולא מעורך SQL ריק.

![ספריית השאילתות השמורות: רשת של שאילתות שניתן להשתמש בהן שוב, גם תבניות מובנות וגם מותאמות אישית](/agenteye/images/queries.png)

*ספריית השאילתות השמורות שלך ב-`/<org-slug>/queries`: תבניות מובנות שישבות ליד השאילתות שהצוות שלך שמר.*

## התחל מתבנית, לא מעמוד ריק

אתה לא צריך לזכור שמות טבלאות או לכתוב SQL מאפס. הספריה נפתחת עם תבניות מובנות לשאלות שהצוותים שואלים הכי הרבה, ישבות ממש ליד השאילתות ששמורות והוסברו ע"י הצוות שלך. בחר אחת שקרובה למה שאתה רוצה ואתה כמעט בחצי הדרך לתשובה.

כל שאילתה שמורה היא בהיקף ארגון ומשותפת, כך שהשאילתות השימושיות שהעמיתים שלך כותבים הופכות גם שלך. תן שם לשאילתה וספק לה תיאור פעם אחת, וכל אחד בארגון שלך יכול למצוא אותה, להריץ אותה, או לצפות בתוצאותיה בלוח המחוונים מאוחר יותר.

מצא את זה ב-`/<org-slug>/queries`.

## התאם אותה והרץ בקומפוזר SQL

פתח כל שאילתה והיא נחתת בקומפוזר SQL, כאשר אתה יכול להתאים אותה ולראות את התשובה מיד: ללא ייצוא, ללא נסיעה הלוך ושוב, ללא ממתנה למישהו אחר.

![קומפוזר שאילתות SQL המריץ שאילתה שמורה, עם סרגל ערכת סכמה ורשת תוצאות חי](/agenteye/images/query-lab.png)

*קומפוזר SQL: השאילתה שלך בצד שמאל, סרגל ערכת סכמה כדי שלעולם לא תנחש שם עמודה, ורשת תוצאות חי למטה.*

- **סרגל ערכת סכמה** מציג את טבלאות הניתוח והעמודות שלהן, כך שאתה יכול לעצב שאילתה ללא ציד שמות שדות.
- **רשת תוצאות חי** מחזירה שורות ברגע שהרצת, כך שאתה חוזר עליה בשניות ולא במשחק של ניחוש וניחוש מחדש.
- **לקריאה בלבד בעיצוב.** שאילתות רצות על חנות האירועים שלך ומאומתות בשרת: רק הצהרות `SELECT` ו-`WITH` מותרות, עם timeout הצהרה וגבול שורות. שאילתה חקירה לא יכולה לעולם לשנות את הנתונים שלך, ואחד שבורח מושמט עבורך.

שמחה עם התוצאה? שמור אותה חזרה לספריה כדי שהצוות כולו יירש אותה, או צפה בפלט שלה בלוח המחוונים כברים קו, עמודה, שטח או עוגה.

## הרץ אותם מהטרמינל, או תן לעוזר לכתוב אותם

אותן שאילתות שמורות עוקבות אחריך לכל מקום שאתה עובד:

- **מהטרמינל.** ה-CLI של `agenteye` מפרט, מריץ ושומר אותן שאילתות, כדי שתוכל להפיל תוצאה לתסריט, לחבור אותה ל-CI, או להעביר אותה לסוכן קידוד.

```bash
agenteye query list                 # אותן שאילתות שמורות, מהטרמינל שלך
agenteye query run errs --arg prod  # הרץ אחת והדפס את השורות (הוסף --json כדי לצנור אותה)
```

  ראה [CLI וסוכנים](/he/agenteye/cli-and-agents) עבור קבוצת הפקודה המלאה.

- **מעוזר ה-AI.** לא בטוח איך לנסח את SQL? שאל את [עוזר ה-AI](/he/agenteye/assistant) בלוח המחוונים באנגלית רגילה והוא ישרטט את השאילתה ויחסוך אותה לספריה שלך עבורך.

הרצת שאילתה שמורה מונעת על ידי ההרשאה `queries:run`, משומרת בנפרד מההרשאות ליצור או למחוק שאילתות, כך שתוכל להעניק גישה לקריאה ללא להרשות לכל אחד לשכתב את הספריה.
