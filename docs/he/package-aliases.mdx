## החבילה הרשמית

החבילה npm הקנונית היא **`failproofai`**:

```bash
npm install -g failproofai
# or
bun add -g failproofai
```

---

## למה אנחנו בעלי שמות הכינויים

Typosquatting היא התקפת שרשרת אספקה נפוצה שבה שחקן זדוני רושם שם חבילה הנמצא במרחק הקשה אחד מחבילה פופולרית. משתמשים חסרי חשד שטועים בהקלדת פקודת ההתקנה מסיימים בהרצת קוד נשלט על ידי התוקף עם גישה מלאה למערכת - בדיוק סוג האיום שFailproof AI מעוצב להגנה נגדו.

כדי לחסל משטח זה, **אנחנו בעלי כל הטעויות כתיב וגרסאות עיצוב נפוצות** של `failproofai` ב-npm. שום שם מהשמות הללו לא יכול להיות רשום על ידי צד שלישי. כל אחד מהם הוא פרוקסי דק המתקין ומעביר לחבילת `failproofai` האמיתית.

## כינויים רשומים

**גרסאות עיצוב** - דרכים שונות לכתוב "failproof ai":

| חבילה | סטטוס |
|---------|--------|
| `failproof` | ✅ פורסם |
| `failproof-ai` | ⏳ בהמתנה לתמיכת npm |
| `fail-proof-ai` | ⏳ בהמתנה לתמיכת npm |
| `failproof_ai` | ⏳ בהמתנה לתמיכת npm |
| `fail_proof_ai` | ⏳ בהמתנה לתמיכת npm |
| `fail-proofai` | ⏳ בהמתנה לתמיכת npm |

**`failprof*` typos** - חסר `o` אחד מ-"proof":

| חבילה | סטטוס |
|---------|--------|
| `failprof` | ✅ פורסם |
| `failprof-ai` | ✅ פורסם |
| `failprofai` | ⏳ בהמתנה לתמיכת npm |
| `fail-prof-ai` | ⏳ בהמתנה לתמיכת npm |
| `failprof_ai` | ⏳ בהמתנה לתמיכת npm |

**`faliproof*` typos** - `a` ו-`i` בחילוף מקום:

| חבילה | סטטוס |
|---------|--------|
| `faliproof` | ✅ פורסם |
| `faliproof-ai` | ✅ פורסם |
| `faliproofai` | ⏳ בהמתנה לתמיכת npm |

> **למה בהמתנה?** מדיניות מניעת ספאם של npm חוסמת שמות שמנורמלים לאותו מחרוזת כחבילה קיימת לאחר הסרת סימנים פונקטואציה וביצוע בדיקות דמיון. יצרנו קשר עם תמיכת npm כדי להזמין שמות אלה לצורכי אנטי-סקוואטינג. הם יופעלו ברגע שיאושרו.

אתה יכול לאמת שכל כינוי פורסם הוא בבעלותנו:

```bash
npm info failproof
# חפש: "ExosphereHost Inc." בשדה maintainers
```

## איך הכינויים עובדים

כל חבילת כינוי:

1. מציינת את `failproofai` כתלות - כך החבילה האמיתית (כולל הגדרת ה-`postinstall` hook שלה) רצה בעת ההתקנה
2. חושפת בינארי שתואם לשם שלה (למשל `failprof-ai`) שמעביר את כל הארגומנטים לבינארי של `failproofai`

הפרוקסי הוא סקריפט Node בשתי שורות; אין ממשק, אין קריאות רשת, ואין איסוף נתונים מעבר למה שעצמו `failproofai` עושה.
