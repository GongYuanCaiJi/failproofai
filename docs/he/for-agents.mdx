הוסף את ההתייחסות המלאה של Failproof AI לסוכן הקוד שלך בפקודה אחת. עובד עם Claude Code, Cursor, Windsurf וכל סוכן אחר התומך בכישרונות.

```bash
npx skills add https://docs.befailproof.ai
```

`npx skills` מזהה אילו סוכנים מותקנים אצלך והוסיף את הכישרון בפורמט הנכון לכל אחד באופן אוטומטי.

## מה הכישרון כולל

| אזור | מה כלול |
|------|--------|
| Policies | שמות הפוליסיות המובנות, סוגי אירועים, פרמטרים, הפעלה/השבתה |
| Custom policies | `customPolicies.add()`, מסננות התאמה, `allow`/`deny`/`instruct` API |
| Context object | `ctx.eventType`, `ctx.toolName`, `ctx.toolInput`, `ctx.session` |
| Configuration | מבנה `policies-config.json`, מיזוג הרחבות, `policyParams` |
| CLI | `failproofai policies --install`, `--uninstall`, `--custom`, הרחבות |
| Dashboard | מציג סשנים, פעילות הפוליסיה, משתני סביבה |
| Architecture | זרימת טיפול ה-Hook, קודי יציאה, חוזה stdin/stdout |

## האם הכישרון שלם?

Mintlify יוצר `llms.txt` מכל העמודים בניווט. התיעוד של Failproof AI מכסה את ה-API המלא - כל פוליסיה, אפשרות ודוגמה כלולות. אם אתה מוצא משהו חסר, המקור נמצא ב-`https://docs.befailproof.ai/llms-full.txt`.

להקשר ממוקד, קשר ישירות לעמוד ספציפי:

```bash
# רק את Custom Policies API
npx skills add https://docs.befailproof.ai/custom-policies

# רק את הפוליסיות המובנות
npx skills add https://docs.befailproof.ai/built-in-policies
```
