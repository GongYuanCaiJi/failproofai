Hook e policy che mantengono i tuoi agenti AI affidabili e in esecuzione autonoma - per **Claude Code**, **OpenAI Codex** e **Agents SDK**.

Gli agenti AI falliscono in modi prevedibili. Eseguono comandi distruttivi, perdono segreti, si allontanano dall'obiettivo, rimangono bloccati in loop, o effettuano push direttamente su main. Se non controllati, i piccoli errori si cascano in fallimenti più grandi.

Failproof AI risolve questo con **policy** - regole che si collegano ad ogni chiamata di strumento dell'agente e possono **rilevare i fallimenti**, **mitigarli** (bloccare, istruire, sanificare) e **avvisarti** quando qualcosa richiede attenzione. Un dashboard locale ti permette di esaminare tutto in seguito.

Tutto viene eseguito localmente. Nessun dato lascia il tuo computer.

## Avvio rapido

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # abilita le policy
failproofai                      # avvia il dashboard
```

Consulta la guida [Inizia](/it/getting-started) per la procedura completa.
