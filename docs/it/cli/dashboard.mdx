---
title: Visualizza sessioni
description: "Avvia la dashboard per sfogliare le sessioni dell'agente e gestire le policy"
---

```bash
failproofai
```

Avvia la dashboard web all'indirizzo `http://localhost:8020`.

## Opzioni

| Flag | Descrizione |
|------|-------------|
| `--port <number>` | Porta su cui restare in ascolto (predefinito: `8020`) |
| `--allowed-origins <origins>` | Host/IP separati da virgole autorizzati ad accedere alle risorse di sviluppo |

Per puntare la dashboard a una cartella di progetto Claude non predefinita, impostare la variabile d'ambiente `CLAUDE_PROJECTS_PATH` all'avvio.

## Esempi

```bash
# Avvia su una porta diversa
failproofai --port 9000

# Usa un percorso di progetti Claude personalizzato tramite variabile d'ambiente
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
