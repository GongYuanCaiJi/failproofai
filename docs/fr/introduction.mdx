Hooks et politiques qui maintiennent vos agents IA fiables et autonomes — pour **Claude Code**, **OpenAI Codex** et l'**Agents SDK**.

Les agents IA échouent de manière prévisible. Ils exécutent des commandes destructrices, laissent fuir des secrets, s'éloignent de leur tâche, se retrouvent coincés dans des boucles, ou poussent directement sur la branche principale. Sans surveillance, de petites défaillances peuvent s'enchaîner et s'aggraver.

Failproof AI résout ce problème grâce aux **politiques** — des règles qui s'accrochent à chaque appel d'outil de l'agent et peuvent **détecter les défaillances**, **les atténuer** (bloquer, instruire, assainir), et **vous alerter** lorsqu'une attention particulière est requise. Un tableau de bord local vous permet de tout passer en revue par la suite.

Tout s'exécute en local. Aucune donnée ne quitte votre machine.

## Démarrage rapide

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # activer les politiques
failproofai                      # lancer le tableau de bord
```
