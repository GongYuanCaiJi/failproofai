Des hooks et des politiques qui rendent vos agents IA fiables et autonomes — pour **Claude Code**, **OpenAI Codex** et l'**Agents SDK**.

Les agents IA échouent de manière prévisible. Ils exécutent des commandes destructrices, divulguent des secrets, s'écartent de leurs tâches, se retrouvent bloqués dans des boucles ou poussent directement vers la branche principale. Laissés sans surveillance, de petites défaillances peuvent s'enchaîner et s'aggraver.

Failproof AI résout ce problème avec des **politiques** — des règles qui s'accrochent à chaque appel d'outil de l'agent et peuvent **détecter les défaillances**, **les atténuer** (bloquer, instruire, assainir) et **vous alerter** lorsqu'une situation nécessite votre attention. Un tableau de bord local vous permet de tout passer en revue par la suite.

Tout s'exécute localement. Aucune donnée ne quitte votre machine.

## Démarrage rapide

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # activer les politiques
failproofai                      # lancer le tableau de bord
```
