---
title: Sitzungen anzeigen
description: "Dashboard starten, um Agent-Sitzungen zu durchsuchen und Richtlinien zu verwalten"
---

```bash
failproofai
```

Startet das Web-Dashboard unter `http://localhost:8020`.

## Optionen

| Flag | Beschreibung |
|------|--------------|
| `--port <number>` | Port, auf dem gelauscht wird (Standard: `8020`) |
| `--projects-path <path>` | Überschreibt den Pfad, unter dem Claude Code-Projektordner gefunden werden |
| `--allowed-origins <origins>` | Kommagetrennte Hosts/IPs, denen der Zugriff auf Entwicklerressourcen erlaubt ist |

## Beispiele

```bash
# Dashboard auf einem anderen Port starten
failproofai --port 9000

# Benutzerdefinierten Projektpfad verwenden
failproofai --projects-path ~/my-claude-projects
```
