---
title: Sitzungen anzeigen
description: "Dashboard starten, um Agent-Sitzungen zu durchsuchen und Richtlinien zu verwalten"
---

```bash
failproofai
```

Startet das Web-Dashboard unter `http://localhost:8020`.

## Optionen

| Flag | Beschreibung |
|------|-------------|
| `--port <number>` | Port, auf dem gehört wird (Standard: `8020`) |
| `--allowed-origins <origins>` | Kommagetrennte Hosts/IPs, die auf Entwicklungsressourcen zugreifen dürfen |

Um das Dashboard auf einen nicht standardmäßigen Claude-Projektordner zu verweisen, setzen Sie beim Start die Umgebungsvariable `CLAUDE_PROJECTS_PATH`.

## Beispiele

```bash
# Auf einem anderen Port starten
failproofai --port 9000

# Benutzerdefinierten Claude-Projektpfad über Umgebungsvariable verwenden
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
