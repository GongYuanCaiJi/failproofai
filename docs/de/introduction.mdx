Hooks und Richtlinien, die Ihre KI-Agenten zuverlässig und autonom am Laufen halten – für **Claude Code**, **OpenAI Codex** und das **Agents SDK**.

KI-Agenten scheitern auf vorhersehbare Weise. Sie führen destruktive Befehle aus, leaken Secrets, verlieren den Fokus, geraten in Endlosschleifen oder pushen direkt auf main. Werden kleine Fehler nicht behoben, schaukeln sie sich hoch.

Failproof AI löst dieses Problem mit **Richtlinien** – Regeln, die sich in jeden Tool-Aufruf eines Agenten einklinken und Fehler **erkennen**, **beheben** (blockieren, anweisen, bereinigen) sowie Sie **benachrichtigen** können, wenn etwas Aufmerksamkeit erfordert. Ein lokales Dashboard ermöglicht es Ihnen, alles im Nachhinein zu überprüfen.

Alles läuft lokal. Keine Daten verlassen Ihren Computer.

## Schnellstart

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # Richtlinien aktivieren
failproofai                      # Dashboard starten
```

Den vollständigen Einstieg finden Sie im [Erste Schritte](/de/getting-started)-Leitfaden.
