Hooks und Richtlinien, die Ihre KI-Agenten zuverlässig und autonom halten – für **Claude Code**, **OpenAI Codex** und das **Agents SDK**.

KI-Agenten scheitern auf vorhersehbare Weisen. Sie führen destruktive Befehle aus, leaken Secrets, verlieren den Fokus, geraten in Endlosschleifen oder pushen direkt auf main. Kleine Fehler können sich unbeaufsichtigt zu größeren Problemen aufschaukeln.

Failproof AI löst dies mit **Richtlinien** – Regeln, die sich in jeden Agenten-Tool-Aufruf einklinken und Fehler **erkennen**, **beheben** (blockieren, anweisen, bereinigen) und Sie **benachrichtigen** können, wenn etwas Aufmerksamkeit erfordert. Ein lokales Dashboard ermöglicht es Ihnen, alles nachträglich zu überprüfen.

Alles läuft lokal. Keine Daten verlassen Ihren Computer.

## Schnellstart

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # Richtlinien aktivieren
failproofai                      # Dashboard starten
```

Den vollständigen Einstieg finden Sie im [Erste-Schritte-Leitfaden](/de/getting-started).
