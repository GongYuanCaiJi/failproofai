Hooks и policies, которые обеспечивают надежность AI-агентов и их автономную работу — для **Claude Code**, **OpenAI Codex** и **Agents SDK**.

AI-агенты отказывают предсказуемым образом. Они выполняют деструктивные команды, утекают секреты, отклоняются от задач, застревают в циклах или делают push прямо в main. Без контроля маленькие ошибки каскадируют.

Failproof AI решает это с помощью **policies** — правил, которые встраиваются в каждый вызов инструмента агента и могут **обнаруживать ошибки**, **смягчать их** (блокировать, давать инструкции, санитизировать) и **алертить вас**, когда что-то требует внимания. Локальная панель мониторинга позволяет вам просмотреть всё впоследствии.

Всё работает локально. Никакие данные не покидают вашу машину.

## Быстрый старт

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # enable policies
failproofai                      # launch the dashboard
```

Полное описание см. в руководстве [Начало работы](/ru/getting-started).
