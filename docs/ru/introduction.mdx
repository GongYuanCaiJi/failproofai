Хуки и политики, которые сохраняют надежность и автономность ваших AI-агентов — для **Claude Code**, **OpenAI Codex** и **Agents SDK**.

AI-агенты отказывают по предсказуемым причинам. Они выполняют деструктивные команды, раскрывают секреты, отвлекаются от задачи, застревают в циклах или направляют изменения прямо в главную ветку. При отсутствии контроля небольшие сбои становятся серьезными проблемами.

Failproof AI решает эту проблему с помощью **политик** — правил, которые перехватывают каждый вызов инструмента агента и могут **выявить сбои**, **смягчить их** (заблокировать, дать инструкцию, санитизировать) и **оповестить вас** о проблемах, требующих внимания. Локальная панель управления позволяет просмотреть все впоследствии.

Все работает локально. Никакие данные не покидают вашу машину.

## Быстрый старт

<CodeGroup>

```bash npm
npm install -g failproofai
```

```bash bun
bun add -g failproofai
```

</CodeGroup>

```bash
failproofai policies --install   # включить политики
failproofai                      # запустить панель управления
```

Полный процесс смотрите в руководстве [Начало работы](/ru/getting-started).
