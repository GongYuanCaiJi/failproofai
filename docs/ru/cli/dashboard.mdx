---
title: Просмотр сессий
description: "Запустите панель управления для просмотра сессий агентов и управления политиками"
---

```bash
failproofai
```

Запускает веб-панель управления по адресу `http://localhost:8020`.

## Параметры

| Флаг | Описание |
|------|----------|
| `--port <number>` | Порт для прослушивания (по умолчанию: `8020`) |
| `--allowed-origins <origins>` | Разделённые запятыми хосты/IP-адреса, разрешённые для доступа к ресурсам разработки |

Чтобы указать папку проекта Claude, отличную от папки по умолчанию, задайте переменную окружения `CLAUDE_PROJECTS_PATH` при запуске.

## Примеры

```bash
# Запуск на другом порту
failproofai --port 9000

# Использование пользовательского пути к проектам Claude через переменную окружения
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
