---
title: Просмотр сессий
description: "Запустите панель управления для просмотра сессий агентов и управления политиками"
---

```bash
failproofai
```

Запускает веб-панель управления на `http://localhost:8020`.

## Параметры

| Флаг | Описание |
|------|-------------|
| `--port <number>` | Порт для прослушивания (по умолчанию: `8020`) |
| `--projects-path <path>` | Переопределить путь, где находятся папки проектов Claude Code |
| `--allowed-origins <origins>` | Разделённые запятыми хосты/IP-адреса, которым разрешен доступ к ресурсам разработки |

## Примеры

```bash
# Запуск на другом порту
failproofai --port 9000

# Использование пользовательского пути к проектам
failproofai --projects-path ~/my-claude-projects
```
