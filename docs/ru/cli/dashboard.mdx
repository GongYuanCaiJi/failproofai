---
title: Просмотр сессий
description: "Запустите панель управления для просмотра сессий агентов и управления политиками"
---

```bash
failproofai
```

Запускает веб-панель управления по адресу `http://localhost:8020`.

## Опции

| Флаг | Описание |
|------|---------|
| `--port <number>` | Порт для прослушивания (по умолчанию: `8020`) |
| `--allowed-origins <origins>` | Разделённые запятыми хосты/IP-адреса, которым разрешён доступ к ресурсам разработки |

Чтобы направить панель управления на папку проекта Claude, не используемую по умолчанию, установите переменную окружения `CLAUDE_PROJECTS_PATH` при запуске.

## Примеры

```bash
# Запуск на другом порту
failproofai --port 9000

# Использование пользовательской папки проектов Claude через переменную окружения
CLAUDE_PROJECTS_PATH=~/my-claude-projects failproofai
```
